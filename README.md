# Spigot Dig

### Version 1.2

Django application showing a summary of the financial information on a chosen US territory. 

## Information shown

1. Gross domestic produce
2. Main industry inside state

### Sources

  ```
Bureau of Labor Statistics

U.S. Bureau of Economic Analysis

  ```
